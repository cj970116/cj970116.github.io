数组去重

这也是一道常见关于数组去重的面试题，在es6中有个简单粗暴的办法，使用new Set()以及Array.from()或者展开运算符（…）

1
2
3
4
5
6
7
8
9

let friuts = ['banana','apple','orange','watermelon','apple']

//方法一
let unqiueFriuts = Array.from(new Set(friuts))
console.log(unqiueFriuts) //['banana','apple','orange',watermelon]

//方法二
let unqiueFriuts2 = [...new Set(friuts)]
console.log(unqiueFriuts2)

替换

日常开发中经常需要替换或者删除一些指定的数据，遇到这种情况一定要想到 Array.protoType.splice这个方法，该方法接收3个参数，第一个参数是开始的索引，第二个参数是需要删除的数量，剩下的就是要添加的值（可以是一个或者多个）

1
2
3

let friuts = ['banana','apple','orange','watermelon','apple']
friuts.splice(0,2,'potato','tomato')
console.log(friuts) //['potato','tomato','orange','watermelon','apple']

清空数组

有时候我们需要清空一个数组，比如用户点击了清空购物车，可以一条一条的删除，但是这样用户体验差，其实一行代码就能搞定，那就是将数组的length设置为0

1
2
3
4

var fruits = [“banana”, “apple”, “orange”, “watermelon”, “apple”, “orange”, “grape”, “apple”];

fruits.length = 0;
console.log(fruits); // returns []

数组转换为对象

有的时候需要将数组转换为对象的格式，使用.map一类的数组迭代方法能达到目的，但是还有更快的方法，前提是你希望对象的key正好就是数组的索引

1
2
3

let fruits = [“banana”, “apple”, “orange”, “watermelon”];
let fruitsObj = { …fruits };
console.log(fruitsObj); // returns {0: “banana”, 1: “apple”, 2: “orange”, 3: “watermelon”, 4: “apple”, 5: “orange”, 6: “grape”, 7: “apple”}

填充数组

创建数组的时候，你有没有遇到过需要填充上默认值的场景，你肯定首先想到的就是循环这个数组。ES6提供了更便捷的.fill方法

1
2

let newArray = new Array(3).fill('1')
console.log(newArray) //['1','1','1','1']

合并数组

说到合并数组，首先想到的就是.concat(),但是今天我们要介绍的是es6展开运算符（…）

1
2
3
4

let friuts = ['apple','banana','orange']
let meat= ['poulyry','beff','fish']
let food = [...friuts,...meat]
console.log(food)   // ['apple','banana','orange','poulyry','beff','fish']

两个数组的交集

1
2
3
4

let numOne = [0,2,4,6,8,8]
let numTwo = [1,2,3,4,5,6]
let commonValues = [...new Set(numOne)].filter(item => numTwo.includes(item))
console.log(commonValues) //[2,4,6]

去除假值

首先，我们要知道假值是什么？在js中，假值有：false，0，''，null，undefined,NAN。现在我们需要找到这些值并将它们移除，可以使用.fliter方法

1
2
3

let mixedArr = [0,'blue','',NAN,9,undefined,false,true]
let trueArr = mixedArr.filter(Boolean)
console.log(trueArr)  //['blue',9,true]

获取数组中的随机值

考察的知识点是随机生成一个值x：X>=0 && X<数组的length

1
2

let colors = ['bule','white','green']
let randomColor = colors[(Math.floor(Math.random()*(colors.length)))]

这里的Math.random是生成一个[0,1)之间的数，Math.floor表示向下取整

lastIndexOf()

很多时候我们查找元素是否存在于某个数组中，经常使用indexOf方法，常常忽略lastIndexOf方法，后者会被使用的一个场景就是，某个数组中有重复的数据

1
2
3

var nums = [1, 5, 2, 6, 3, 5, 2, 3, 6, 5, 2, 7];
var lastIndex = nums.lastIndexOf(5);
console.log(lastIndex); // returns 9

求和

对数组中的所有元素进行求和，哈哈，又是一道如数家珍的面试题。答案也是很多，条条大道通罗马，这里使用的是reduce，reduce方法是很值得学习的知识点，用处很多。

1
2
3

var nums = [1, 5, 2, 6];
var sum = nums.reduce((x, y) => x + y);
console.log(sum); // returns 14
